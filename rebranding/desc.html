Rebranding and Web Banner Ads

This was the final project for my Fundamentals of Visual Literacy (Fall 2019) class. The objective was to redesign a well-known company's website and create web banners (advertisements) of predetermined sizes. I chose to rebrand Coca-Cola. The website redesign was done in HTML and CSS, and the web banners are simple images.

SKILLS LEARNED/FOCUSED

  • Technical ability:
    • Web technologies
    • Digital media

TOOLS USED

  • Caret (text and code editor)
  • LibreOffice Draw (image and flowchart editor)
  • GIMP (image editor)
  • Google Chrome (web browser)

DESIGN CHOICES

The website redesign is supposed to resemble a full bottle of Coca-Cola with the color scheme of off-black, red and white. For the web-banners, I used the red-and-white color scheme similar to the printed label on a bottle of the well-known beverage.

CHALLENGES

The primary challenge in this project was learning to use GIMP. GIMP is very similar to Adobe Photoshop, except some of its features are different along with the interface. At one point my experimentation with the color settings caused an image in the editor to be locked into a hideous olive-green color scheme.

WHERE TO FROM HERE?

  • Using LibreOffice Draw to create detailed wireframes of site concepts
  • Using GIMP to create and edit logos/images for websites
